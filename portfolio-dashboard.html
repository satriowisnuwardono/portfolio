Dashboard

Dibuat untuk memenuhi kebutuhan instansi untuk mengintegrasikan data antar SKPD.

  • Client: Departemen Jakarta Satu
  • Completed: Berjalan
  • Versi: 1.0
    • Lebih Lanjut

Saya mendapatkan tugas untuk melakukan pengembangan di bagian Backend. Disini saya mendapatkan tugas untuk membuat RESTful API dari sistem yang akan dikembangkan.

Sistem ini dibuat dan dirancang untuk mengintegrasikan SKPD di jakarta khususnya agar dapat berbagi dan pakai data sesuai dengan kebutuhan mereka. Pada sistem ini pula bertujuan untuk memberikan kemudahan dalam melakukan kelola data, dimana data tersebut nantinya dapat pula di konsumsi oleh masyarakat umum tentunya dengan kebijakan yang telah di atur oleh tiap-tiap SKPD yang berbagi datanya.
